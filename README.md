# Fort-in-Sahyadri

**Tourism Sahyadri Fort Website**

Created a static website that enables the user to browse through the Visitor Information, Images, Map and Videos of popular Fort.  

**Technologies Used Technologies Used**: HTML, CSS, Bootstrap Flexbox, JavaScript  
    Developed mobile-first layout using Bootstrap flex and CSS box model  
    Implemented Navigation via Carousal point  
    Designed and developed a responsive, informative website showcasing the historical forts of the Sahyadri mountain range in Maharashtra.
